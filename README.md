# topPage ssr nextJS


##Installation dev


1. Install 

   ```bash
   yarn
   ```

2. Run

   ```bash
   yarn dev
   ```

## Build project

1. build 

   ```
   yarn build
   ```

2. Run

   ```bash
   yarn start
   ```
   
## Run docker

1. Change directory in Dockerfile 
    
   ```bash
    WORKDIR /yourPath/
   ```

2. Build 
   ```bash
   docker build -t top-page
   ```
3. Change   
    ```bash
    docker-compose.yml
    ```
4. Run 
   ```bash
   docker-compose up -d
   ```
